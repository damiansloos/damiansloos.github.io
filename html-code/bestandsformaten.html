bestandsformaten

Deze pagina gaat over de bestandsformaten in een computer.

Zoals op de pagina van het binaire stelsel staat gebruikt een computer bits, enen en nullen om informatie en gegevens te kunnen beheren.
Deze nullen en enen zijn in de computer de bits. Het zijn energiecellen die een lading vasthouden, een zo'n energiecel is een bit.
Acht van deze nullen en enen, of een combinatie daartussen die achter elkaar staan wordt een byte genoemd.
Dit zijn dus 8 bits achter elkaar. Die een byte maken.
Nu is een byte echt heel weinig informatie, zo kan het een enkel lettertje, cijfertje of tekentje zijn.
In de computer zitten dus heel heel heel veel van deze bytes.
Nu heb je vast wel een van "mb" of "gb" gehoord. Dit is de opslag van je computer, waarmee je apps kan downloaden.
Die "mb" en "gb" zijn heel erg veel van deze bytes.
Voor elke factor duizend kot er een letter voor, dus 1000 bytes is een kilo-byte.
Dit gaat snel uit de hand lopen met nullen, dus zal ik de cijfers geven in de wetenschappelijke notatie.
Een kilo-byte is dus 1.0 x 10^3 bytes, de 3 bij 10^3 is het aantal nullen.
Een kb = 1.0 x 10^3, een mb = 1.0 x 10^6, een gb = 1.0 x 10^9, op deze manier kan je wel even doorgaan.
